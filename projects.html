Medical Store Management System

The Medical Store Management System is a web-based application designed to streamline and automate the management of a medical store's operations. Built with Node.js, Express, MongoDB, and Mongoose, this system provides intuitive CRUD (Create, Read, Update, Delete) functionalities for core entities, including Customer, Medicine, Pharmacist, and Inventory. Users can add, update, or delete medicines with details like Medicine Name, Medicine ID, Price, and Quantity, ensuring accurate stock management. Additionally, the system enables efficient customer tracking and pharmacist records management, promoting operational efficiency and improving the overall user experience in a medical retail setting.

Rock-Paper-Scissor Game

The console-based game developed in Python offers an engaging experience where users can challenge the computer in a classic game of Rock-Paper-Scissors. Players can select their moves, and the program uses randomization to determine the computer's choice, providing a fair and unpredictable gameplay experience. This project showcases fundamental programming concepts such as loops, conditionals, and user input handling, while also emphasizing game logic and interaction. The simplicity of the game, combined with its interactive nature, makes it an excellent introduction to game development in Python.

Treasure Hunt Game

In this text-based adventure game created with Python, players embark on an exciting quest to uncover hidden treasures by solving a series of intriguing clues. The game immerses players in a rich narrative, where they navigate through various locations, encounter challenges, and make choices that impact their journey. Leveraging Python's control structures and data management capabilities, the game offers a dynamic experience that encourages critical thinking and problem-solving. With engaging storytelling and interactive gameplay, this project not only demonstrates proficiency in Python but also showcases creativity in game design, making it a captivating experience for players of all ages.
